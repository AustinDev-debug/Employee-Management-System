# Employee Management System

A C# console application that allows users to manage employee records through a menu-driven interface.

## Features

* Add new employees
* List all employees
* Search for an employee by name
* Update an employee's salary
* Remove an employee from the system
* Input validation for both text and numeric values

## Concepts Used

* C#
* Classes and Objects
* Multiple Class Files
* Lists (`List<T>`)
* Methods
* Loops (`for` and `foreach`)
* Switch Statements
* Input Validation

## What I Learned

This project helped me practice organizing a C# application into multiple files, working with custom objects stored in a `List<T>`, searching and updating objects, removing items from a list, and creating a menu-driven console application.

## Possible Future Improvements

* Load employee data when the application starts.

* Prevent duplicate employee IDs.
* Improve searching with case-insensitive and exact-match options.
